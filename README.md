# clojure-fractal

Small project for drawing Fractals with Clojure.

## Usage

Run in your shell:

    lein deps && lein install && lein repl
   
Run in your repl:

    (use 'clojure-fractal.core)
   
    (start fractal)

### Draw Your Fractal

    (draw fractal)

### Export Your Fractal to Image File

    (export fractal)
